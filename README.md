## 特性

- 使用Dart语言开发
- Material3主题
- 强智教务账号密码登录
- 课表显示
- 当前上课及下一节课高亮
- 部分账号支持自动重新登录
- 无任何广告
- 更多特性开发中

## 安装

访问本项目的 [Releases](https://github.com/YiQiuYes/schedule/releases)
页面下载安装包。

- Android 用户可以直接通过 `Releases` 页面下载最新安装包来安装

- IOS 用户下载 `ipa` 安装包后，通过 [TrollStore](https://github.com/opa334/TrollStore) 来安装
